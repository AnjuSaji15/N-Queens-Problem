# N-Queens Problem Solver

## Overview
The N-Queens Problem Solver is a graphical user interface (GUI) application built using Python's Tkinter library. This application solves the N-Queens problem using a backtracking algorithm and displays the solutions on a chessboard.

## Features
- Accepts any number of queens (N) greater than or equal to 4.
- Visualizes the placement of queens on a chessboard.
- Allows users to navigate through multiple solutions.
- User-friendly interface with clear instructions and error handling.
